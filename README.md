## 📖 Project Description

QuickShow is a modern web-based application designed to simplify movie and event booking. The platform provides users with a seamless experience to explore trending movies, check details, and simulate ticket booking.

The system is designed with a focus on **user experience, performance, and real-time dynamic data integration**, making it comparable to modern OTT and booking platforms.

---

## 🚀 Key Features

* 🎬 **Trending Movies (Top 10 Today)**
* 🔍 **Search Movies Functionality**
* 🎥 **Movie Details Page (Poster, Rating, Overview)**
* ▶️ **Trailer Integration (YouTube Embed)**
* 🎟️ **Ticket Booking Simulation**
* 📱 **Responsive & Modern UI (Netflix-style design)**
* ⚡ **Fast Performance with Optimized API Calls**
